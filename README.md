# Linked List Implementation
This project includes 100% test coverage, with over 40 unit tests. Project belongs to App Acdemy.

Contains two classes, Node and LinkedList. 

 ## Node class:
  - Attributes:  
     - `value`
     - `next`
   
 ## LinkedList class:
  - Attributes:
    - `head`
    - `tail` 
    - `length` 
    
  - Methods:
  
    - `addToTail(value)`
    - `addToHead(value)`
    - `removeTail()`
    - `removeHead()`
    - `set(value)`
    - `get(index)`
    - `contains(value)`
    - `insert(index)`
    - `remove(index)`
    - `size()`
